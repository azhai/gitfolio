// Features
功能完备，开箱即用
覆盖代码托管全流程，从仓库管理到项目协作，一站式解决。
🔐

用户认证和授权

基于 JWT 的安全认证体系，支持注册、登录、权限分级管理

📦

Git 仓库管理

完整的仓库 CRUD、分支与标签管理、提交历史浏览、文件查看

📝

Issue 跟踪系统

标签分类、人员指派、评论讨论，支持多维度筛选和状态管理

🔀

Pull Request 管理

代码审查、合并、关闭、重开，完整的 PR 工作流支持

✅

任务管理系统

排期规划、附件管理、关联 Issue、状态流转、时间追踪

🔗

提交自动关联

提交消息自动解析关联 Issue/PR/Task，支持 Fixes #123 等语法

📝

提交详情与比较

签名验证、文件变更查看、差异对比，深入理解每一次提交

⏱️

任务时间追踪

内置计时器、时间汇总统计，精准把控项目进度

👥

团队/组织管理

创建团队、管理成员、权限控制，支持多人协作开发

🏷️

里程碑管理

项目阶段规划与跟踪，关联 Issue 和 PR，把控版本节奏

📊

仓库统计和活动流

代码统计、提交活动、贡献者分析，项目健康一目了然

💻

代码片段管理

创建和分享代码片段，支持多语言语法高亮

🔄

仓库同步

GitHub 镜像同步，定时拉取代码和 Issue 数据，保持远程同步

⭐

Star 和 Watch

收藏感兴趣的仓库，关注仓库动态，不错过任何更新

🚀

Release 版本管理

版本发布与标签管理，从 GitHub 同步 Release 信息
