GitFolio Core

Git 仓库管理核心系统，提供完整的版本控制和协作功能。

功能特性

  • 完整的 Git 仓库管理
  • Issue 跟踪系统
  • 合并请求和代码审查
  • 团队协作功能
  • Wiki 文档系统

快速开始

# 克隆仓库
git clone https://gitfolio.com/ryan/gitfolio-core.git

# 安装依赖
npm install

# 启动开发服务器
npm run dev

技术栈

  • Node.js - 运行环境
  • Express - Web 框架
  • MongoDB - 数据库
  • Redis - 缓存
